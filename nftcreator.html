NFT Creator

In a shameless attempt to make some cash money, and jump on the NFT collections bandwagon, I've created this tool to create NFTs. The NFT Creator can be downloaded from its GitHub repository. This is the first time I've posted something on GitHub to actually be used by other people so, apologies if it's not pretty. The basics are you have a set of folders in a particular structure (it must be correct for the files to land in the right place!) then you have a colorise script which colorises the elements and a combine script that combines the elements into unique images.
