Loneliness Reimagined is an interactive I'm working on at the moment, and should be up soon. It's one of many projects that explore the similar non-zero-sum ideas to NEUTROPIA if you'd like to learn more you can follow the link to explore that project.

As for Loneliness Reimagined, I'm planning on building an exchange system where different coloured agents can share 'artworks' which express something about themselves, which others can consume, and develop an 'understanding' of the different colours so that you can connect, but only by giving something of yourself and putting yourself out there.
