Loneliness Reimagined is an interactive I'm working on at the moment, and should be up soon. It's one of many projects that explore the similar non-zero-sum ideas as NEUTROPIA if you'd like to learn more you can follow the link to explore that project.

As for this Loneliness Reimagined, I'm planning on build an exchange system where different coloured agents can share 'artworks' which express something about them, which others can consume, and develop an 'understanding' of the different colours so that the can make friends with each other - so you can connect, but only by giving something over yourself and putting yourself out there.
